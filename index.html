About me

I am a young boy with 16 years old who loves development and learn new things, i am very dedicated and i learn fast.
I started liking development when i was 12 when i had the interest and built a Discord bot using NodeJS (same year where i was OBMEP bronze medalist). With the time i lost a little the focus on development, but i never stopped practicing. In 2020 with the emergence of the pandemic we were all affected by the lockdown, and with much more time at home, i started for deepen learn more of development. Keeping in NodeJS i practiced developing some Rest APIs using Express, SQL and MongoDB for hobby, also i built some Rest APIs using PHP and SQL, and did some Discord automations with Python too. During this period i meet a friend that in 2021 noticing the front-end evolution, he started to learn more about it and encouraged me, where it was the kick off for me begin learning the famous web development triad: HTML, CSS and JavaScript. At the same time i learned the triad, i also learned Git. I always keep learning and trying new things, at this moment i am learning ReactJS and TypeScript.
